# Capstone
Capstone project base on Travel booking system 

Description
**Tropic Dodo** is a comprehensive travel booking system designed to simplify trip planning in Mauritius. It offers an all-in-one platform where users can browse and book accommodations, activities, and transportation seamlessly. With features like personalized recommendations, secure payment integration, and real-time availability updates, Tropic Dodo provides a user-friendly experience tailored for both local and international travelers. The system also incorporates local insights to promote authentic Mauritian experiences, making it a reliable travel companion for exploring paradise.


For Tropic Dodo, a travel booking system, you can consider the following languages and technologies:

1. Frontend (User Interface):
HTML5: For building the structure of your pages.
CSS3: To style the UI and ensure it’s visually appealing.
JavaScript: To add interactivity and dynamic content.
Bootstrap: A CSS framework to quickly design a responsive layout.
React.js or Vue.js: For building the front-end with more interactive and dynamic user interfaces.
2. Backend (Server-Side):
PHP: If you prefer working with traditional server-side technologies for dynamic web applications.
3. Database:
MySQL or PostgreSQL: Relational databases to store and manage bookings, user data, and product listings efficiently
4. APIs:
Google Maps API: For location-based services like maps and trip routes.
Payment Gateway APIs (Stripe, PayPal): To facilitate secure payments for bookings.
5. Other Tools:
Git/GitHub: For version control and collaboration.
