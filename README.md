# Capstone
Capstone project base on Travel booking system 

Description
**Tropic Dodo** is a comprehensive travel booking system designed to simplify trip planning in Mauritius. It offers an all-in-one platform where users can browse and book accommodations, activities, and transportation seamlessly. With features like personalized recommendations, secure payment integration, and real-time availability updates, Tropic Dodo provides a user-friendly experience tailored for both local and international travelers. The system also incorporates local insights to promote authentic Mauritian experiences, making it a reliable travel companion for exploring paradise.
